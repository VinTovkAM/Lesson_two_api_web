# Скрипт конвертер валют

[Exchange Rate API](https://www.exchangerate-api.com/docs/overview) это API для работы с валютами из разных стран мира.

## Установка

Проверьте установлен ли Python версии 3.10.6 на вашем компьютере. Если нет, то установите его по [ссылке](https://www.python.org/downloads/release/python-3106/).

Установите зависимости через команду:
```bash
pip install -r requirements.txt
```

## Окружение

Используйте виртуальное окружение [venv](https://docs.python.org/3/library/venv.html) для корректной работы проекта.


## Переменные окружения

Не забудьте добавить файл типа `.env` в директорию с вашим проектом и указать ваш токен для работы с API:

```
EXCHANGERATE_API_KEY = = 'your_token'
```

## Использование

В терминале вызовите скрипт с аргументами:
```bash
python main.py -b YOUR_CURRENCY_BASE_CODE -t YOUR_CURRENCY_TARGER_CODE -a AMOUNT 
```

Например, можно узнать курс евро к доллару, написав:
```bash
python main.py -b EUR -t USD -a 1 
```

Скрипт отобразит конвертированную валюту, если при запросе не произошло ошибки.

_По умолчанию, скрипт отобразит курс рубля к доллару с количеством валюты ─ 1000_ 

## Дополнительная информация

1. Прежде, чем использовать скрипт, пожалуйста, удостовертесь в правильности и стабильности ваше интернет соединения.

2. Скрипт использует `response.raise_for_status()` для обработки HTTP ошибок.

3. При ошибке смотрите информацию о ней в терминале.
